Using shift

Using cargo fuzz

cargo install cargo-fuzz

Run

Run fuzzer for the parser, for 15 minutes.

cd fuzz
rustup default nightly
cargo +nightly fuzz run --sanitizer none parser -- -only_ascii=1 -max_total_time=900 -timeout=5
